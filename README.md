### 2. 🔍 **Fake News & Bias Detection** - Capstone Project 2
**Classification & NLP**

Detects fake news and biased content from online forums using machine learning algorithms on 12,999 news articles.

- **Tech Stack:** Python, Scikit-learn, XGBoost, LightGBM, NLP
- **Models:** Logistic Regression, Decision Trees, Random Forest, SVM, Gradient Boosting
- **Challenge:** Handling class imbalance and missing data (32.5%)
- **Key Metrics:** Accuracy, Precision, Recall, F1-Score, ROC-AUC
